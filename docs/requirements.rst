Requirements
============
 
The requirements necessary to use the AUN TELA MIS Project are:
 
- **python3** and **pip3**
- **virtualenv and virtualenvwrapper**
- **Firefox** (to use Selenium's Webdriver in functional Tests)
- **GNU gettext** (to use Internationalization)
